# MultiCLI 项目优势

## 核心优势

- **编排者主导的工作流**：单一协调者负责规划、分发、集成和总结任务
- **多 CLI 协同执行**：Claude/Codex/Gemini 工作者并行或串行执行任务
- **功能契约与验收标准**：确保前后端保持一致对齐
- **依赖感知调度**：基于 DAG 的执行保证前置依赖按序完成
- **文件级锁定**：防止同一文件的冲突编辑
- **快照与回滚支持**：任务出错时可恢复文件
- **内置验证阶段**：编译/Lint/测试钩子作为质量门禁
- **上下文压缩**：在长会话中保留记忆同时节省 Token
- **审查循环**：最终集成前进行自检和互检
- **UI 可追溯性**：完整的任务时间线、输出和工具调用面板
