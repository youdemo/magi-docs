## 故障排除

### 如果测试失败

1. **检查编译**
   ```bash
   npm run compile
   ```
   确保没有编译错误

2. **检查项目根目录**
   测试会在项目根目录创建 `.multicli/knowledge` 目录

3. **查看详细错误**
   测试输出会显示具体的失败原因

### 常见问题

**Q: 测试运行很慢？**
A: 知识库初始化需要扫描项目文件，第一次运行会比较慢

**Q: 某些测试失败？**
A: 检查错误消息，可能是权限问题或文件访问问题

**Q: 如何只运行特定测试？**
A: 目前需要修改测试文件，注释掉不需要的测试
