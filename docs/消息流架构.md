## 一、产品目标与核心价值

### 1.1 产品定位

MultiCLI 是一个 **多 AI CLI 编排平台**，核心价值在于：

1. **统一入口** - 用户通过单一界面与多个 AI CLI 工具交互
2. **智能编排** - 自动分解任务，分配给最合适的 CLI 执行
3. **协同工作** - 多个 CLI 并行/串行协作完成复杂任务
4. **透明可控** - 用户能清晰看到每个 CLI 的工作状态和输出

## 二、当前架构问题分析

### 2.1 消息流路径过长

```
当前架构（6 层）:

CLI Process → InteractiveSession → SessionManager → PersistentSessionAdapter
            → CLIAdapterFactory → WebviewProvider → Webview

问题：
- 每层格式不同，需要反复转换
- 职责不清，多层都在做"事件转发"
- 调试困难，消息丢失难以定位
```

### 2.2 缺乏统一消息模型

```typescript
// 当前：每层定义自己的格式
InteractiveSession:  { cli, role, chunk }
SessionManager:      { cli, role, chunk }
CLIAdapterFactory:   { type, chunk, source, adapterRole }
WebviewProvider:     ExtensionToWebviewMessage (10+ 种类型)

// 问题：
// 1. 字段命名不一致 (cli vs type, role vs adapterRole)
// 2. 语义重叠 (source vs adapterRole)
// 3. 缺少关键信息 (渲染目标、消息类型)
```

### 2.3 CLI 响应解析不统一

```typescript
// 当前：只有一个解析器，假设所有 CLI 格式相同
parseStreamJsonResponse() {
  // 只处理 Claude 的 stream-json 格式
  // Codex 没有指定输出格式
  // Gemini 的 stream-json 可能格式不同
}
```

### 2.4 渲染目标决策分散

```typescript
// 当前：渲染目标在多处决定
WebviewProvider:
  - orchestratorStreamBuffer (编排者 → 对话面板)
  - handleCliOutput (Worker → ???)

// 问题：
// 1. 没有统一的决策逻辑
// 2. target 字段使用不一致
// 3. 相同内容可能重复显示
```

## 三、新架构设计

### 3.1 设计原则

1. **单一数据模型** - 所有消息使用统一的 `UnifiedMessage` 格式
2. **职责分离** - 解析、转换、路由、渲染各司其职
3. **管道模式** - 消息经过标准化管道处理
4. **可扩展性** - 新增 CLI 只需实现解析器接口

### 3.2 新架构概览


## 四、统一消息模型

### 4.1 核心类型定义

```typescript
/**
 * 统一消息模型
 * 所有消息在系统内部都使用这个格式
 */
interface UnifiedMessage {
  /** 消息唯一标识 */
  id: string;

  /** 消息时间戳 */
  timestamp: number;

  /** 消息来源 */
  source: MessageSource;

  /** 消息内容 */
  content: MessageContent;

  /** 任务上下文 */
  context: MessageContext;

  /** 渲染指令 */
  rendering: RenderingDirective;
}

/**
 * 消息来源
 */
interface MessageSource {
  /** 角色类型 */
  role: 'orchestrator' | 'worker' | 'system' | 'user';

  /** CLI 类型（如果来自 CLI） */
  cli?: CLIType;

  /** Worker ID（如果来自 Worker） */
  workerId?: string;

  /** 显示名称（用于 UI 展示） */
  displayName: string;
}

/**
 * 消息内容
 */
interface MessageContent {
  /** 内容类型 */
  type: ContentType;

  /** 主要内容 */
  text: string;

  /** 格式 */
  format: 'plain' | 'markdown' | 'json' | 'diff';

  /** 附加数据（根据 type 不同而不同） */
  data?: ContentData;
}

/**
 * 内容类型枚举
 */
type ContentType =
  | 'text'           // 普通文本
  | 'thinking'       // 思考过程
  | 'code'           // 代码块
  | 'diff'           // 文件差异
  | 'error'          // 错误信息
  | 'question'       // 询问
  | 'progress'       // 进度更新
  | 'plan'           // 执行计划
  | 'summary'        // 任务总结
  | 'tool_use'       // 工具调用
  | 'tool_result'    // 工具结果
  | 'file_change';   // 文件变更

/**
 * 任务上下文
 */
interface MessageContext {
  /** 会话 ID */
  sessionId: string;

  /** 任务 ID */
  taskId?: string;

  /** 子任务 ID */
  subTaskId?: string;

  /** 请求 ID（用于关联请求和响应） */
  requestId?: string;
}

/**
 * 渲染指令
 */
interface RenderingDirective {
  /** 渲染目标 */
  target: RenderTarget;

  /** 是否流式输出 */
  streaming: boolean;

  /** 是否追加到上一条消息 */
  append: boolean;

  /** 优先级 */
  priority: 'high' | 'normal' | 'low';

  /** 是否可折叠 */
  collapsible: boolean;
}

/**
 * 渲染目标
 */
type RenderTarget =
  | 'thread'      // 对话面板
  | 'cli'         // CLI 输出面板
  | 'both'        // 两者都显示
  | 'silent';     // 不显示（仅记录）
```

## 五、CLI 解析器设计

### 5.1 解析器接口

```typescript
/**
 * CLI 响应解析器接口
 * 每种 CLI 实现自己的解析器
 */
interface ICLIParser {
  /** CLI 类型 */
  readonly cliType: CLIType;

  /**
   * 解析原始输出
   * @param raw 原始字节/字符串
   * @param buffer 当前缓冲区（用于处理不完整的 JSON）
   * @returns 解析结果
   */
  parse(raw: string, buffer: string): ParseResult;

  /**
   * 检测是否是询问
   */
  detectQuestion(chunk: ParsedChunk): QuestionInfo | null;

  /**
   * 提取 token 使用信息
   */
  extractTokenUsage(chunks: ParsedChunk[]): TokenUsage | null;
}

/**
 * 解析结果
 */
interface ParseResult {
  /** 解析出的块 */
  chunks: ParsedChunk[];
  /** 剩余未解析的内容（不完整的 JSON 等） */
  remainder: string;
  /** 是否完成（收到结束标记） */
  complete: boolean;
}

/**
 * 解析后的块
 */
interface ParsedChunk {
  /** 块类型 */
  type: ChunkType;
  /** 主要内容 */
  content: string;
  /** 元数据 */
  metadata: Record<string, unknown>;
  /** 原始数据（用于调试） */
  raw?: string;
}

/**
 * 块类型
 */
type ChunkType =
  | 'text'              // 普通文本
  | 'thinking'          // 思考过程
  | 'tool_use_start'    // 工具调用开始
  | 'tool_use_input'    // 工具调用参数
  | 'tool_result'       // 工具结果
  | 'error'             // 错误
  | 'result'            // 最终结果
  | 'usage'             // token 使用
  | 'question'          // 询问
  | 'unknown';          // 未知类型
```

### 5.2 Claude 解析器实现

```typescript
/**
 * Claude CLI 解析器
 * 处理 stream-json 格式输出
 */
class ClaudeParser implements ICLIParser {
  readonly cliType: CLIType = 'claude';

  parse(raw: string, buffer: string): ParseResult {
    const combined = buffer + raw;
    const lines = combined.split('\n');
    const chunks: ParsedChunk[] = [];
    let remainder = '';

    for (let i = 0; i < lines.length; i++) {
      const line = lines[i].trim();
      if (!line) continue;

      // 最后一行可能不完整
      if (i === lines.length - 1 && !combined.endsWith('\n')) {
        remainder = line;
        continue;
      }

      try {
        const json = JSON.parse(line);
        const chunk = this.parseJsonLine(json);
        if (chunk) {
          chunks.push(chunk);
        }
      } catch {
        // 非 JSON 行，作为普通文本处理
        chunks.push({
          type: 'text',
          content: line,
          metadata: {},
          raw: line,
        });
      }
    }

    const complete = chunks.some(c => c.type === 'result');
    return { chunks, remainder, complete };
  }

  private parseJsonLine(json: any): ParsedChunk | null {
    switch (json.type) {
      case 'assistant':
        return this.parseAssistantMessage(json);

      case 'content_block_start':
        return this.parseContentBlockStart(json);

      case 'content_block_delta':
        return this.parseContentBlockDelta(json);

      case 'tool_use':
        return {
          type: 'tool_use_start',
          content: json.name || '',
          metadata: { tool: json.name, input: json.input },
        };

      case 'tool_result':
        return {
          type: 'tool_result',
          content: json.content || '',
          metadata: { toolUseId: json.tool_use_id },
        };

      case 'result':
        return {
          type: 'result',
          content: json.result || '',
          metadata: {
            cost: json.cost_usd,
            duration: json.duration_ms,
            usage: json.usage,
          },
        };

      case 'error':
        return {
          type: 'error',
          content: json.error?.message || json.message || 'Unknown error',
          metadata: { code: json.error?.code },
        };

      default:
        return null;
    }
  }

  private parseAssistantMessage(json: any): ParsedChunk | null {
    const content = json.message?.content;
    if (!Array.isArray(content)) return null;

    const textBlocks = content
      .filter((b: any) => b.type === 'text')
      .map((b: any) => b.text)
      .join('');

    if (!textBlocks) return null;

    return {
      type: 'text',
      content: textBlocks,
      metadata: {},
    };
  }

  private parseContentBlockStart(json: any): ParsedChunk | null {
    const block = json.content_block;
    if (!block) return null;

    if (block.type === 'thinking') {
      return {
        type: 'thinking',
        content: block.thinking || '',
        metadata: { index: json.index },
      };
    }

    if (block.type === 'tool_use') {
      return {
        type: 'tool_use_start',
        content: block.name || '',
        metadata: { tool: block.name, id: block.id, index: json.index },
      };
    }

    return null;
  }

  private parseContentBlockDelta(json: any): ParsedChunk | null {
    const delta = json.delta;
    if (!delta) return null;

    if (delta.type === 'thinking_delta') {
      return {
        type: 'thinking',
        content: delta.thinking || '',
        metadata: { index: json.index, delta: true },
      };
    }

    if (delta.type === 'text_delta') {
      return {
        type: 'text',
        content: delta.text || '',
        metadata: { index: json.index, delta: true },
      };
    }

    if (delta.type === 'input_json_delta') {
      return {
        type: 'tool_use_input',
        content: delta.partial_json || '',
        metadata: { index: json.index, delta: true },
      };
    }

    return null;
  }

  detectQuestion(chunk: ParsedChunk): QuestionInfo | null {
    // Claude 的询问通常在 tool_use 中
    if (chunk.type === 'tool_use_start') {
      const tool = chunk.metadata.tool as string;
      if (tool === 'ask_user' || tool === 'request_permission') {
        return {
          type: 'permission',
          question: chunk.content,
          options: chunk.metadata.options as string[] | undefined,
        };
      }
    }

    // 检查文本中的询问模式
    if (chunk.type === 'text') {
      const text = chunk.content;
      if (/\[Y\/n\]/i.test(text) || /\[y\/N\]/i.test(text)) {
        return {
          type: 'confirmation',
          question: text,
        };
      }
    }

    return null;
  }

  extractTokenUsage(chunks: ParsedChunk[]): TokenUsage | null {
    const resultChunk = chunks.find(c => c.type === 'result');
    if (!resultChunk) return null;

    const usage = resultChunk.metadata.usage as any;
    if (!usage) return null;

    return {
      inputTokens: usage.input_tokens || 0,
      outputTokens: usage.output_tokens || 0,
      cacheReadTokens: usage.cache_read_input_tokens,
      cacheWriteTokens: usage.cache_creation_input_tokens,
    };
  }
}
```

```typescript
class CodexParser implements ICLIParser {
  readonly cliType: CLIType = 'codex';

  parse(raw: string, buffer: string): ParseResult {
    const combined = buffer + raw;
    const chunks: ParsedChunk[] = [];

    // Codex 输出是纯文本，按行处理
    const lines = combined.split('\n');
    let remainder = '';

    for (let i = 0; i < lines.length; i++) {
      const line = lines[i];

      // 最后一行可能不完整
      if (i === lines.length - 1 && !combined.endsWith('\n')) {
        remainder = line;
        continue;
      }

      const chunk = this.parseLine(line);
      if (chunk) {
        chunks.push(chunk);
      }
    }

    // Codex 没有明确的结束标记，通过进程退出判断
    return { chunks, remainder, complete: false };
  }

  private parseLine(line: string): ParsedChunk | null {
    const trimmed = line.trim();
    if (!trimmed) return null;

    // 检测工具调用模式
    if (trimmed.startsWith('Running:') || trimmed.startsWith('Executing:')) {
      return {
        type: 'tool_use_start',
        content: trimmed.replace(/^(Running|Executing):\s*/, ''),
        metadata: { tool: 'bash' },
      };
    }

    // 检测文件操作
    if (trimmed.startsWith('Writing to') || trimmed.startsWith('Creating')) {
      const match = trimmed.match(/(?:Writing to|Creating)\s+(.+)/);
      return {
        type: 'tool_use_start',
        content: match?.[1] || trimmed,
        metadata: { tool: 'write_file' },
      };
    }

    if (trimmed.startsWith('Reading')) {
      const match = trimmed.match(/Reading\s+(.+)/);
      return {
        type: 'tool_use_start',
        content: match?.[1] || trimmed,
        metadata: { tool: 'read_file' },
      };
    }

    // 检测错误
    if (trimmed.startsWith('Error:') || trimmed.startsWith('error:')) {
      return {
        type: 'error',
        content: trimmed.replace(/^[Ee]rror:\s*/, ''),
        metadata: {},
      };
    }

    // 普通文本
    return {
      type: 'text',
      content: line,
      metadata: {},
    };
  }

  detectQuestion(chunk: ParsedChunk): QuestionInfo | null {
    if (chunk.type !== 'text') return null;

    const text = chunk.content;

    // Codex 的确认提示
    if (/proceed\?/i.test(text) || /continue\?/i.test(text)) {
      return {
        type: 'confirmation',
        question: text,
      };
    }

    if (/\(y\/n\)/i.test(text)) {
      return {
        type: 'confirmation',
        question: text,
      };
    }

    return null;
  }

  extractTokenUsage(_chunks: ParsedChunk[]): TokenUsage | null {
    // Codex 不提供 token 使用信息
    return null;
  }
}
```

```typescript
/**
 * Gemini CLI 解析器
 * 处理 Gemini 的 stream-json 格式
 */
class GeminiParser implements ICLIParser {
  readonly cliType: CLIType = 'gemini';

  parse(raw: string, buffer: string): ParseResult {
    const combined = buffer + raw;
    const lines = combined.split('\n');
    const chunks: ParsedChunk[] = [];
    let remainder = '';

    for (let i = 0; i < lines.length; i++) {
      const line = lines[i].trim();
      if (!line) continue;

      if (i === lines.length - 1 && !combined.endsWith('\n')) {
        remainder = line;
        continue;
      }

      try {
        const json = JSON.parse(line);
        const chunk = this.parseJsonLine(json);
        if (chunk) {
          chunks.push(chunk);
        }
      } catch {
        // 非 JSON 行
        chunks.push({
          type: 'text',
          content: line,
          metadata: {},
        });
      }
    }

    const complete = chunks.some(c => c.type === 'result');
    return { chunks, remainder, complete };
  }

  private parseJsonLine(json: any): ParsedChunk | null {
    // Gemini 的 JSON 格式可能与 Claude 不同
    // 这里需要根据实际 Gemini CLI 输出格式调整

    if (json.text) {
      return {
        type: 'text',
        content: json.text,
        metadata: {},
      };
    }

    if (json.functionCall) {
      return {
        type: 'tool_use_start',
        content: json.functionCall.name,
        metadata: {
          tool: json.functionCall.name,
          args: json.functionCall.args,
        },
      };
    }

    if (json.functionResponse) {
      return {
        type: 'tool_result',
        content: JSON.stringify(json.functionResponse.response),
        metadata: { name: json.functionResponse.name },
      };
    }

    if (json.error) {
      return {
        type: 'error',
        content: json.error.message || 'Unknown error',
        metadata: { code: json.error.code },
      };
    }

    if (json.usageMetadata) {
      return {
        type: 'usage',
        content: '',
        metadata: {
          promptTokens: json.usageMetadata.promptTokenCount,
          candidatesTokens: json.usageMetadata.candidatesTokenCount,
          totalTokens: json.usageMetadata.totalTokenCount,
        },
      };
    }

    return null;
  }

  detectQuestion(chunk: ParsedChunk): QuestionInfo | null {
    if (chunk.type === 'tool_use_start') {
      const tool = chunk.metadata.tool as string;
      if (tool === 'askUser' || tool === 'confirmAction') {
        return {
          type: 'permission',
          question: chunk.content,
        };
      }
    }

    return null;
  }

  extractTokenUsage(chunks: ParsedChunk[]): TokenUsage | null {
    const usageChunk = chunks.find(c => c.type === 'usage');
    if (!usageChunk) return null;

    return {
      inputTokens: usageChunk.metadata.promptTokens as number || 0,
      outputTokens: usageChunk.metadata.candidatesTokens as number || 0,
    };
  }
}
```

```typescript
/**
 * 消息转换器
 * 将 ParsedChunk 转换为 UnifiedMessage

class MessageTransformer {
  private renderingRules: RenderingRuleEngine;

  constructor() {
    this.renderingRules = new RenderingRuleEngine();
  }

  /**
   * 转换解析后的块为统一消息
   */
  transform(
    chunk: ParsedChunk,
    source: MessageSourceInfo,
    context: MessageContext
  ): UnifiedMessage {
    const id = this.generateMessageId();
    const timestamp = Date.now();

    // 1. 构建消息来源
    const messageSource = this.buildSource(source);

    // 2. 构建消息内容
    const content = this.buildContent(chunk);

    // 3. 决定渲染指令
    const rendering = this.renderingRules.decide(messageSource, content);

    return {
      id,
      timestamp,
      source: messageSource,
      content,
      context,
      rendering,
    };
  }

  /**
   * 批量转换（用于流式输出合并）
   */
  transformBatch(
    chunks: ParsedChunk[],
    source: MessageSourceInfo,
    context: MessageContext
  ): UnifiedMessage[] {
    // 合并连续的相同类型块
    const merged = this.mergeConsecutiveChunks(chunks);
    return merged.map(chunk => this.transform(chunk, source, context));
  }

  private buildSource(info: MessageSourceInfo): MessageSource {
    const displayNames: Record<string, Record<CLIType, string>> = {
      orchestrator: {
        claude: '编排者',
        codex: '编排者',
        gemini: '编排者',
      },
      worker: {
        claude: 'Claude Worker',
        codex: 'Codex Worker',
        gemini: 'Gemini Worker',
      },
    };

    return {
      role: info.role,
      cli: info.cli,
      workerId: info.workerId,
      displayName: displayNames[info.role]?.[info.cli] || info.cli,
    };
  }

  private buildContent(chunk: ParsedChunk): MessageContent {
    const typeMapping: Record<ChunkType, ContentType> = {
      'text': 'text',
      'thinking': 'thinking',
      'tool_use_start': 'tool_use',
      'tool_use_input': 'tool_use',
      'tool_result': 'tool_result',
      'error': 'error',
      'result': 'summary',
      'usage': 'progress',
      'question': 'question',
      'unknown': 'text',
    };

    const formatMapping: Record<ChunkType, MessageContent['format']> = {
      'text': 'markdown',
      'thinking': 'plain',
      'tool_use_start': 'plain',
      'tool_use_input': 'json',
      'tool_result': 'plain',
      'error': 'plain',
      'result': 'markdown',
      'usage': 'json',
      'question': 'markdown',
      'unknown': 'plain',
    };

    return {
      type: typeMapping[chunk.type] || 'text',
      text: chunk.content,
      format: formatMapping[chunk.type] || 'plain',
      data: chunk.metadata,
    };
  }

  private mergeConsecutiveChunks(chunks: ParsedChunk[]): ParsedChunk[] {
    if (chunks.length === 0) return [];

    const merged: ParsedChunk[] = [];
    let current = { ...chunks[0] };

    for (let i = 1; i < chunks.length; i++) {
      const chunk = chunks[i];

      // 合并连续的相同类型文本块
      if (chunk.type === current.type &&
          (chunk.type === 'text' || chunk.type === 'thinking')) {
        current.content += chunk.content;
      } else {
        merged.push(current);
        current = { ...chunk };
      }
    }

    merged.push(current);
    return merged;
  }

  private generateMessageId(): string {
    return `msg_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`;
  }
}
```

```typescript
 * 根据消息来源和内容类型决定渲染方式

class RenderingRuleEngine {
  private rules: RenderingRule[] = [];

  constructor() {
    this.initializeDefaultRules();
  }

  /**
   * 决定渲染指令
   */
  decide(source: MessageSource, content: MessageContent): RenderingDirective {
    // 按优先级查找匹配的规则
    for (const rule of this.rules) {
      if (this.matchRule(rule, source, content)) {
        return rule.directive;
      }
    }

    // 默认规则
    return {
      target: 'thread',
      streaming: false,
      append: false,
      priority: 'normal',
      collapsible: false,
    };
  }

  private matchRule(
    rule: RenderingRule,
    source: MessageSource,
    content: MessageContent
  ): boolean {
    if (rule.sourceRole && rule.sourceRole !== source.role) {
      return false;
    }
    if (rule.contentType && rule.contentType !== content.type) {
      return false;
    }
    if (rule.cli && rule.cli !== source.cli) {
      return false;
    }
    return true;
  }
```

```typescript
/**
 * 消息路由器
 * 根据渲染目标分发消息到对应处理器
 */
class MessageRouter extends EventEmitter {
  private handlers: Map<RenderTarget, MessageHandler[]> = new Map();

  constructor() {
    super();
    this.initializeHandlers();
  }

  /**
   * 路由消息
   */
  route(message: UnifiedMessage): void {
    const target = message.rendering.target;

    if (target === 'both') {
      // 同时发送到 thread 和 cli
      this.dispatch('thread', message);
      this.dispatch('cli', message);
    } else if (target === 'silent') {
      // 静默处理（仅记录日志）
      this.logMessage(message);
    } else {
      this.dispatch(target, message);
    }
  }

  /**
   * 批量路由
   */
  routeBatch(messages: UnifiedMessage[]): void {
    for (const message of messages) {
      this.route(message);
    }
  }

  private dispatch(target: RenderTarget, message: UnifiedMessage): void {
    const handlers = this.handlers.get(target) || [];
    for (const handler of handlers) {
      try {
        handler.handle(message);
      } catch (error) {
        console.error(`[MessageRouter] Handler error for ${target}:`, error);
      }
    }

    // 发出事件供外部监听
    this.emit('message', { target, message });
  }

  private logMessage(message: UnifiedMessage): void {
    console.log(`[Silent] ${message.source.displayName}: ${message.content.text.slice(0, 100)}...`);
  }

  /**
   * 注册处理器
   */
  registerHandler(target: RenderTarget, handler: MessageHandler): void {
    if (!this.handlers.has(target)) {
      this.handlers.set(target, []);
    }
    this.handlers.get(target)!.push(handler);
  }

  private initializeHandlers(): void {
    // 初始化默认处理器
    this.handlers.set('thread', []);
    this.handlers.set('cli', []);
    this.handlers.set('silent', []);
  }
}

/**
 * 消息处理器接口
 */
interface MessageHandler {
  handle(message: UnifiedMessage): void;
```

```typescript
/**
 * 对话面板处理器
 * 处理显示在对话面板的消息
 */
class ThreadHandler implements MessageHandler {
  private streamingMessages: Map<string, StreamingMessage> = new Map();
  private onMessage: (message: ThreadMessage) => void;

  constructor(onMessage: (message: ThreadMessage) => void) {
    this.onMessage = onMessage;
  }

  handle(message: UnifiedMessage): void {
    if (message.rendering.streaming && message.rendering.append) {
      // 流式消息：追加到现有消息
      this.handleStreamingMessage(message);
    } else {
      // 完整消息：直接发送
      this.sendMessage(message);
    }
  }

  private handleStreamingMessage(message: UnifiedMessage): void {
    const streamKey = this.getStreamKey(message);
    let streaming = this.streamingMessages.get(streamKey);

    if (!streaming) {
      // 创建新的流式消息
      streaming = {
        id: message.id,
        source: message.source,
        context: message.context,
        chunks: [],
        startTime: Date.now(),
      };
      this.streamingMessages.set(streamKey, streaming);

      // 发送流开始事件
      this.onMessage({
        type: 'stream_start',
        id: streaming.id,
        source: message.source,
        context: message.context,
      });
    }

    // 追加内容
    streaming.chunks.push(message.content.text);

    // 发送流更新事件
    this.onMessage({
      type: 'stream_chunk',
      id: streaming.id,
      chunk: message.content.text,
    });
  }

  /**
   * 完成流式消息
   */
  completeStream(source: MessageSource, context: MessageContext): void {
    const streamKey = `${source.role}:${source.cli}:${context.taskId}`;
    const streaming = this.streamingMessages.get(streamKey);

    if (streaming) {
      // 发送流完成事件
      this.onMessage({
        type: 'stream_end',
        id: streaming.id,
        fullContent: streaming.chunks.join(''),
      });

      this.streamingMessages.delete(streamKey);
    }
  }

  private sendMessage(message: UnifiedMessage): void {
    this.onMessage({
      type: 'message',
      id: message.id,
      source: message.source,
      content: message.content,
      context: message.context,
      timestamp: message.timestamp,
    });
  }

  private getStreamKey(message: UnifiedMessage): string {
    return `${message.source.role}:${message.source.cli}:${message.context.taskId}`;
  }
}

interface StreamingMessage {
  id: string;
  source: MessageSource;
  context: MessageContext;
  chunks: string[];
  startTime: number;
}

type ThreadMessage =
  | { type: 'message'; id: string; source: MessageSource; content: MessageContent; context: MessageContext; timestamp: number }
  | { type: 'stream_start'; id: string; source: MessageSource; context: MessageContext }
  | { type: 'stream_chunk'; id: string; chunk: string }
  | { type: 'stream_end'; id: string; fullContent: string };
```

```typescript
        const path = data?.path as string || text;
        return `[${action}] ${path}`;

      case 'error':
        return `❌ ${text}`;

      default:
        return text;
    }
  }
```

```typescript
            messageId: message.id,
            chunk: message.chunk,
            target: 'thread',
          },
        });
        break;

      case 'stream_end':
        this.send({
          type: 'streamEvent',
          payload: {
            phase: 'complete',
            messageId: message.id,
            fullContent: message.fullContent,
            target: 'thread',
          },
        });
        break;
    }
  }

  /**
   * 发送 CLI 输出
   */
  sendCLIOutput(output: CLIOutput): void {
    this.send({
      type: 'cliOutput',
      payload: {
        id: output.id,
        cli: output.cli,
        outputType: output.type,
        content: output.content,
        timestamp: output.timestamp,
        collapsible: output.collapsible,
      },
    });
  }

  /**
   * 发送询问请求
   */
  sendQuestion(question: QuestionRequest): void {
    this.send({
      type: 'questionRequest',
      payload: question,
    });
  }

  /**
   * 发送状态更新
   */
  sendStatusUpdate(status: StatusUpdate): void {
    this.send({
      type: 'statusUpdate',
      payload: status,
    });
  }

  private send(message: ExtensionToWebviewMessage): void {
    if (this.isReady) {
      this.webview.postMessage(message);
    } else {
      this.messageQueue.push(message);
    }
  }

  private flushQueue(): void {
    for (const message of this.messageQueue) {
      this.webview.postMessage(message);
    }
    this.messageQueue = [];
  }

  private mapRole(role: MessageSource['role']): 'user' | 'assistant' | 'system' {
    switch (role) {
      case 'user':
        return 'user';
      case 'system':
        return 'system';
      default:
        return 'assistant';
    }
  }
}
```

### 9.1 管道组装

```typescript
/**
 * 消息处理管道
 * 将所有组件组装成完整的消息处理流程
 */
class MessagePipeline {
  private parserFactory: CLIParserFactory;
  private transformer: MessageTransformer;
  private router: MessageRouter;
  private uiBridge: UIBridge;

  // 解析缓冲区（每个 CLI 独立）
  private parseBuffers: Map<string, string> = new Map();

  constructor(webview: vscode.Webview) {
    this.parserFactory = new CLIParserFactory();
    this.transformer = new MessageTransformer();
    this.router = new MessageRouter();
    this.uiBridge = new UIBridge(webview);

    this.setupHandlers();
  }

  private setupHandlers(): void {
    // 注册对话面板处理器
    const threadHandler = new ThreadHandler((msg) => {
      this.uiBridge.sendThreadMessage(msg);
    });
    this.router.registerHandler('thread', threadHandler);

    // 注册 CLI 输出面板处理器
    const cliHandler = new CLIOutputHandler((output) => {
      this.uiBridge.sendCLIOutput(output);
    });
    this.router.registerHandler('cli', cliHandler);
  }

  /**
   * 处理 CLI 原始输出
   * 这是整个管道的入口点
   */
  processRawOutput(
    raw: string,
    cli: CLIType,
    role: 'orchestrator' | 'worker',
    context: MessageContext,
    workerId?: string
  ): void {
    const bufferKey = `${cli}:${role}`;

    // 1. 获取解析器
    const parser = this.parserFactory.getParser(cli);

    // 2. 获取当前缓冲区
    const buffer = this.parseBuffers.get(bufferKey) || '';

    // 3. 解析原始输出
    const parseResult = parser.parse(raw, buffer);

    // 4. 更新缓冲区
    this.parseBuffers.set(bufferKey, parseResult.remainder);

    // 5. 转换为统一消息
    const sourceInfo: MessageSourceInfo = { role, cli, workerId };
    const messages = this.transformer.transformBatch(
      parseResult.chunks,
      sourceInfo,
      context
    );

    // 6. 路由消息
    this.router.routeBatch(messages);

    // 7. 检测询问
    for (const chunk of parseResult.chunks) {
      const question = parser.detectQuestion(chunk);
      if (question) {
        this.handleQuestion(question, cli, role, context);
      }
    }
  }

  /**
   * 处理完成事件
   */
  processComplete(
    cli: CLIType,
    role: 'orchestrator' | 'worker',
    context: MessageContext
  ): void {
    const bufferKey = `${cli}:${role}`;

    // 清空缓冲区
    const remainder = this.parseBuffers.get(bufferKey);
    if (remainder) {
      // 处理剩余内容
      const parser = this.parserFactory.getParser(cli);
      const parseResult = parser.parse(remainder + '\n', '');

      const sourceInfo: MessageSourceInfo = { role, cli };
      const messages = this.transformer.transformBatch(
        parseResult.chunks,
        sourceInfo,
        context
      );
      this.router.routeBatch(messages);
    }

    this.parseBuffers.delete(bufferKey);

    // 通知流完成
    this.uiBridge.sendStatusUpdate({
      type: 'phase',
      phase: 'complete',
    });
  }

  private handleQuestion(
    question: QuestionInfo,
    cli: CLIType,
    role: 'orchestrator' | 'worker',
    context: MessageContext
  ): void {
    this.uiBridge.sendQuestion({
      id: `q_${Date.now()}`,
      source: role === 'orchestrator' ? '编排者' : `${cli} Worker`,
      cli,
      question: question.question,
      options: question.options,
    });
  }

  /**
   * 标记 UI 就绪
   */
  setUIReady(): void {
    this.uiBridge.setReady();
  }
}

interface MessageSourceInfo {
  role: 'orchestrator' | 'worker';
  cli: CLIType;
  workerId?: string;
}
```

```typescript
    // 监听 CLI 适配器工厂的输出事件
    this.cliFactory.on('output', ({ type, chunk, source, adapterRole }) => {
      if (!this.messagePipeline) return;

      const context: MessageContext = {
        sessionId: this.currentSessionId,
        taskId: this.currentTaskId,
      };

      this.messagePipeline.processRawOutput(
        chunk,
        type,
        adapterRole || 'worker',
        context
      );
    });

    // 监听响应完成事件
    this.cliFactory.on('response', ({ type, source, adapterRole }) => {
      if (!this.messagePipeline) return;

      const context: MessageContext = {
        sessionId: this.currentSessionId,
        taskId: this.currentTaskId,
      };

      this.messagePipeline.processComplete(
        type,
        adapterRole || 'worker',
        context
      );
    });
  }
}
```

## 十一、总结

### 11.1 架构对比

| 维度 | 当前架构 | 新架构 |
|------|----------|--------|
| **层数** | 6 层 | 5 层 |
| **消息格式** | 每层不同 | 统一 UnifiedMessage |
| **解析逻辑** | 单一解析器 | 每 CLI 独立解析器 |
| **渲染决策** | 分散在多处 | 集中在 RenderingRuleEngine |
| **可扩展性** | 需要修改多处 | 只需添加解析器 |
| **可测试性** | 难以单元测试 | 每层可独立测试 |

### 11.2 核心改进

1. **统一消息模型** - 所有消息使用 `UnifiedMessage` 格式，消除格式转换混乱
2. **解析器工厂** - 每种 CLI 有独立解析器，正确处理各自的输出格式
3. **渲染规则引擎** - 集中管理渲染决策，规则清晰可配置
4. **消息路由器** - 根据渲染目标自动分发，职责单一
5. **管道模式** - 消息流经标准化管道，易于调试和扩展

### 11.3 预期收益

- **正确性** - 不同 CLI 的输出能被正确解析和显示
- **一致性** - 消息来源、渲染位置有统一的决策逻辑
- **可维护性** - 每层职责清晰，修改影响范围可控
- **可扩展性** - 新增 CLI 只需实现解析器接口
- **可测试性** - 每个组件可独立单元测试
