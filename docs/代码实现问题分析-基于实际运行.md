场景B（不同SubTask，修改同一文件）:
```
时间线:
T0: SubTask-1 createSnapshot('types.ts', 'codex', 'task-1')  // 快照A
T1: SubTask-1 开始修改 types.ts
T2: SubTask-2 createSnapshot('types.ts', 'gemini', 'task-2')  // 快照B（基于T1修改后的文件！）
T3: SubTask-1 完成修改
T4: SubTask-2 完成修改

- task-1 的快照A = 原始文件
- task-2 的快照B = task-1修改后的文件（错误！）
```

❌ **数据完整性问题**

**根本原因**:
- 快照创建时机: **任务执行中**（而非计划时）
- 并行任务: SubTask-2在SubTask-1**修改文件后**创建快照
- 快照内容: SubTask-2的快照包含了SubTask-1的修改

**实际风险**:
- 如果任务并行执行，第二个任务的快照是"脏"的
- 回滚会丢失第一个任务的修改

### 4. 文件锁与任务调度的实际协调

检查 TaskDependencyGraph 的实际实现：

```typescript
// task-dependency-graph.ts:75-100
addTask(task: SubTaskDef, targetFiles?: string[]): void {
  // 1. 检测文件冲突
  const filesForTask = targetFiles || task.targetFiles || [];
  const conflicts: string[] = [];

  for (const file of filesForTask) {
    const owner = this.fileOwnerMap.get(file);
    if (owner && owner !== task.id) {
      conflicts.push(file);
    }
  }

  // 2. 自动添加依赖
  if (conflicts.length > 0) {
    for (const file of conflicts) {
      const owner = this.fileOwnerMap.get(file);
      if (owner && !task.dependencies.includes(owner)) {
        task.dependencies.push(owner);  // ✅ 自动串行化
      }
    }
  }

  // 3. 死锁检测
  if (this.detectCycle(task)) {
    throw new Error(`循环依赖检测: ${task.id}`);
  }
}
```

## 实际修复建议

### 修复1: 上下文自动迁移（P1）

```typescript
class ContextManager {
  addMessage(message, applyTruncation = true) {
    this.immediateContext.push(message);

    // 检查是否需要迁移
    if (this.immediateContext.length > this.maxMessages) {
      // 🆕 提取被移除的消息
      const toRemove = this.immediateContext.slice(0, -this.maxMessages);

      // 🆕 迁移到 Memory
      for (const msg of toRemove) {
        if (this.shouldMigrateToMemory(msg)) {
          this.migrateToMemory(msg);
        }
      }

      // 清理即时上下文
      const systemMessages = this.immediateContext.filter(m => m.role === 'system');
      const otherMessages = this.immediateContext.filter(m => m.role !== 'system');
      this.immediateContext = [
        ...systemMessages,
        ...otherMessages.slice(-this.maxMessages)
      ];
    }
  }

  private shouldMigrateToMemory(message: ContextMessage): boolean {
    // 用户消息、关键决策、错误信息应迁移
    return message.role === 'user' ||
           message.content.includes('决定') ||
           message.content.includes('错误') ||
           message.content.includes('完成');
  }

  private migrateToMemory(message: ContextMessage): void {
    if (!this.sessionMemory) return;

    if (message.role === 'user') {
      // 提取任务描述
      this.sessionMemory.addImportantContext(
        `用户请求: ${message.content.substring(0, 200)}`
      );
    } else {
      // 提取关键结论
      const keyPhrases = this.extractKeyPhrases(message.content);
      if (keyPhrases.length > 0) {
        this.sessionMemory.addImportantContext(keyPhrases.join('; '));
      }
    }
  }
}
```
